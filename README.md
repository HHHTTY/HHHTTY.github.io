# Haitao Yu Academic Homepage

This repository is a pure static GitHub Pages site for `https://HHHTTY.github.io`.

## Structure

- `index.html`: academic homepage content.
- `assets/css/styles.css`: responsive styling.
- `assets/js/main.js`: small progressive enhancements.
- `assets/img/`: profile photo, favicon, and social preview image.
- `.nojekyll`: ensures GitHub Pages serves static assets directly.

## Local preview

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Deployment

Push this folder to the GitHub repository `HHHTTY/HHHTTY.github.io`. GitHub Pages will publish it automatically from the default branch.
